# PG Management System

A role-based PG (Paying Guest) Management Web Application.

## Tech Stack
- Python (Flask)
- Supabase (PostgreSQL + Storage)
- HTML, CSS, JavaScript

## Features
- Admin & Tenant login
- Rent payment tracking with proof upload
- Maintenance request system
- Vacating & deposit settlement workflow

## Run Locally
```bash
pip install -r requirements.txt
python app.py
